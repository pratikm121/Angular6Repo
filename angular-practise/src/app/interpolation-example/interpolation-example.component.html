Interpolation Example 1

This Example shows the usage of interpolation in Angular. Interpolation in Angular is represented by Double Curly braces

Hi {{name}} {{message}}

To find lenght of a string 'Pratik' = {{name.length}}

To convert value of a string 'Pratik' to lowercase = {{name.toLowerCase()}}

To convert value of a string 'Pratik' to Uppercase = {{name.toUpperCase()}}
